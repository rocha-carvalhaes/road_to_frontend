tag afeta todas as tags do tipo p { color: red; }
.classe afeta todos os elementos com essa classe .destaque { font-weight: bold; }
#id afeta um elemento específico #titulo { font-size: 24px; }

o box model:

o box model define como cada elemento é renderizado no layout. ele é composto pelas seguintes estruturas

  • content: o conteúdo do elemento
  • padding: espaço interno
  • border: borda ao redor do elemento
  • margin: espaço externo ao redor do elemento
